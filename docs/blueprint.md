# **App Name**: O Bruxo Nexus

## Core Features:

- User Authentication: Secure account creation and login system supporting email/password and social providers.
- Campaign Management: Tools for GMs to create campaigns, invite players, and manage campaign settings.
- Interactive Character Sheets: Dynamic character sheets with integrated dice rolling, real-time stat updates, inventory management, and spellcasting functionality.
- Combat Tracker: GM-exclusive tool for managing encounters, tracking initiative, HP, conditions, and adding monsters.
- Virtual Tabletop (VTT): Customizable grid, map uploads, token movement, Fog of War, and drawing tools for immersive gameplay.
- Integrated Dice Roller: Dice roller embedded within the chat interface for easy and transparent rolls.
- Descriptive Generator: AI-powered tool to generate descriptions for environments, items, or creatures, maintaining the tone of the game world. The tool makes inferences about when it might be helpful to include additional details from the game's Library.
- Soundboard: A tool with various sound effects and ambient sounds to enhance the game's atmosphere.
- Generation Area: A suite of AI-powered tools to generate items, loot, NPCs, rooms, maps, spells, etc., for game masters. The tool makes inferences about when it might be helpful to include additional details from the game's Library.
- Wiki Area: For lore, atlas, npcs databases, etc...

## Style Guidelines:

- Primary color: Deep violet (#9400D3), evocative of magic.
- Background color: Dark gray (#282828), a desaturated violet hue, creates a focused atmosphere.
- Accent color: Electric yellow (#FFEA00), analogous to violet, will provide contrast for interactive elements.
- Headline font: 'Playfair', serif, for headlines and small amounts of text; body font: 'PT Sans', sans-serif, for longer text.
- Fantasy-themed icons for abilities, items, and actions.
- Intuitive layout with clear separation between GM-View and Player-View.
- Subtle animations for dice rolls, stat changes, and combat actions.
- Overall aesthetic: Galaxy background with glares of sun and universe star stuff. Modern UI/UX concepts with creative animations and movement.
